# Linear Algebra

## Matrix

### Real Symmetric Matrix

1. $A^{T} = A$
2. Eigen vectors are orthogonal
3. Eigen values are real

### Real anti-symmetric Matrix (Skew-Symmetric)

1. $A^{T} = -A$
2. Eigen vectors are orthogonal
3. Eigen values are all imaginary and appear in pair
4. Skew-symetric matrices of odd dimension are singular (a row and a column o zero)
